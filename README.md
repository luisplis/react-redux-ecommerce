# Eccomerce en React con Redux Toolkit

Redux Toolkit para gestionar el estado en toda tu aplicación de tal manera que pueda ser accedido por cualquier componente sin seguir la jerarquía entre los componentes. Construirás una aplicación simple de Comercio Electrónico usando React y Redux Toolkit, donde mostrarás una lista de productos con un botón de "Agregar al Carrito" para cada producto, mostrarás los artículos añadidos al carrito y permitirás a los usuarios eliminar artículos del carrito. Toda esta información estará disponible globalmente en toda la aplicación usando Redux Toolkit.

- Integrar componentes de React con Redux para la gestión del estado en una una interfaz de usuario cohesiva.

- Implementar características básicas de Comercio Electrónico, como agregar artículos al carrito, eliminar artículos del carrito y vaciar el carrito.
